Download the Git at https://git.scm.com/download/win Download the 64-bit or the 32-bit (check your computer) Install the Git Open the Git Bash mkdir git-tutorial (this create a new folder named git-tutorial) Git Bash is recommended to display where branch you are working on You can use linux command in Git Bash like 'ls' etc., 1. git init (to initialized a git folder inside your working folder, setting git in project) 2. git status (to check the there are changes if you have added a new file or if you edited/modified a file) 3. git add index.html (add the index.html file) 4. git rm --cached index.html (to delete the added file) 5. git commit -m "Added a new file index.html" (before we push we must commit first) 6. git push (to push the changes made) To practice it create a repository at Github.com Some recommendations from Github.com: Create a new repository on the command line 1. git init 2. git add README.md 3. git commit -m "First Commit" 4. git branch -M master 5. git remote add origin https://github.com/Canon2/git-tutorial.git 6. git push -u origin master Some recommendations from Github.com: Push an existing repository from the command line 1. git remote add origin https://github.com/Canon2/git-tutorial.git 2. git branch -M master 3. git push -u origin master 1. git add .
